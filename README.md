# furigana4md

A Python script for adding furigana to Japanese Markdown text used in Obsidian.

The generated syntax is compatible with
the [Markdown Furigana Plugin](https://github.com/steven-kraft/obsidian-markdown-furigana) for Obsidian.

Forked from the original [furigana4epub](https://github.com/Mumumu4/furigana4epub) and modified to work with Markdown
files by Claude Sonnet 4.6.
